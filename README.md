# MindfulMe - Mental Health Platform

A production-ready Mental Health Platform with multi-modal AI.

## Features
- Multi-step Onboarding with PHQ-9 and GAD-7 assessments
- Mental Health Index calculation
- Voice Journal with vocal biometrics analysis
- Text Analysis with real-time sentiment detection
- Predictive Analytics with proactive wellness alerts
- JWT Authentication with Biometric support
- End-to-end encryption for sensitive data

## Quick Start

### Backend
```bash
cd backend && npm install && npm run dev
```

### ML Service
```bash
cd ml-service && pip install -r requirements.txt && uvicorn main:app --port 8000
```

### Frontend
```bash
cd frontend && npm install && npx react-native run-ios
```
